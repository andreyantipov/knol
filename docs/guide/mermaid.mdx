# Mermaid in `.mdx`

`meta.txt` treats `.mdx` the same as `.md` in the viewer (frontmatter
shows as-is, since we don't parse MDX components). Mermaid fences work
identically.

## Graph

```mermaid
graph TD
    Root[meta.txt root] --> Bin[bin/meta.ts]
    Root --> Src[src/]
    Root --> Ui[ui/]
    Src --> Server[server.ts]
    Src --> Acp[acp.ts]
    Src --> Assets[assets.ts<br/>auto-generated]
    Ui --> App[App.tsx]
    Ui --> DocContent[doc-content.tsx]
    DocContent --> Mermaid[lib/mermaid.ts]
```
